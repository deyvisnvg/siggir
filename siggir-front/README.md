### <div align="center">FRONTEND</div>

# Requerimientos

- Versión Node

   ```bash
   $ node -v | node --version
   v20.12.1

   $ npm -v | npm --version
   v10.8.3
   ```

- Versión React

   ```bash
   v18.3.1
   ```

# Instalación

   ```bash
   $ git clone https://github.com/deyvisnvg/siggir-front.git
   $ cd siggir-front
   ```

# Empezando

- Instalar paquetes

    ```bash
    npm i 
    ```

- Ejecute el servidor de desarrollo:

    ```bash
    npm run dev
    ```
